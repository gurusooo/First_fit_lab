# First_fit_lab
№10 - реализация алгоритма First Fit для управления памятью

Реализован полноценный менеджер памяти с алгоритмом First Fit
Техническая реализация:
* Структура блоков: заголовок (size, is_free, next, prev)
* Алгоритм First Fit: ищет первый подходящий свободный блок
* Слияние блоков: автоматическое при освобождении
* Выравнивание: кратно 8 байтам
* Защита от ошибок: проверка double-free, нулевых размеров

Алгоритм First Fit был протестирован по 5 направлениям:
1. Базовое выделение и освобождение - успешно,
2. Повторное использование освобождённой памяти - успешно,
3. Фрагментация и слияние блоков - успешно,
4. Граничные случаи (нулевой размер, переполнение) - успешно,
5. Нагрузочное тестирование (100 аллокаций) - успешно!
